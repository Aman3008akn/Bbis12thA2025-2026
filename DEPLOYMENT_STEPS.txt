# Deployment Steps

## 1. Prepare GitHub Repository

```bash
# Navigate to project directory
cd "c:\Users\ankit\bbis 12thA"

# Initialize git
git init

# Add all files
git add .

# Create initial commit
git commit -m "Initial commit: 12th A Memories Website with Admin Dashboard"
```

## 2. Create GitHub Repository

- Go to https://github.com/new
- Repository name: `12th-a-memories`
- Description: `12th A Class Memories Website with Admin Dashboard`
- Choose Public or Private
- Click "Create repository"

## 3. Push to GitHub

```bash
git remote add origin https://github.com/YOUR_USERNAME/12th-a-memories.git
git branch -M main
git push -u origin main
```

## 4. Deploy on Netlify

### Method 1: Connect GitHub (EASIEST)

1. Go to https://app.netlify.com
2. Sign up / Login (can use GitHub account)
3. Click "New site from Git"
4. Select GitHub as provider
5. Choose your repository
6. Build settings:
   - Build command: (leave empty)
   - Publish directory: (leave empty)
7. Click "Deploy site"

### Method 2: Netlify CLI

```bash
# Install CLI globally
npm install -g netlify-cli

# Login to Netlify (opens browser)
netlify login

# Deploy to production
netlify deploy --prod
```

### Method 3: Drag & Drop

1. Go to https://app.netlify.com/drop
2. Drag project folder to drop zone
3. Wait for deployment

## What netlify.toml Does

```toml
[build]
  command = "npm run build"          # Build command
  publish = "."                      # Publish current directory
  
[[redirects]]
  from = "/*"
  to = "/index.html"                # SPA routing
  status = 200
  
[[headers]]
  for = "/*"
  [headers.values]
    Cache-Control = "..."            # Performance
    X-Content-Type-Options = "..."   # Security
    X-XSS-Protection = "..."         # Security
```

## Automatic Deployments

After connecting GitHub:
- Every push to `main` automatically deploys
- Previous versions saved for rollback
- Deployment logs available in Netlify dashboard

## Troubleshooting

### Site not loading
- Check netlify.toml syntax
- Verify all files committed
- Clear browser cache (Ctrl+Shift+Del)

### Admin page shows 404
- This is normal for SPA - netlify.toml handles it
- Admin loads at `/admin.html` via redirect

### Supabase not connecting
- Verify credentials in supabase-config.js
- Check browser console for errors
- Ensure Supabase project is active

### Slow deployment
- First deploy may take 5-10 minutes
- Subsequent deploys faster
- Check build logs in Netlify dashboard
